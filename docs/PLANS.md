# Планы развития

Планируемые улучшения и новые возможности ИИ-Агента.

## Мультимодальный агент

Расширение возможностей агента для работы с разными типами входных данных:

- **Видящий** — обработка изображений (скриншоты форм, отчёты, диаграммы)
- **Слышащий** — распознавание и обработка аудио (голосовые команды, записи)
- **Принимающий файлы** — загрузка документов (PDF, Word, Excel), таблиц для анализа и извлечения данных

Требует поддержки мультимодальных моделей (vision, audio) в провайдерах и расширения формата сообщений диалога.

## Веб-поиск

Возможность поиска информации в интернете для дополнения контекста:

- Интеграция с поисковыми API
- Использование при недостатке данных в метаданных или для актуальной информации (курсы, нормативы)
- Контроль доступа и фильтрация источников для корпоративного использования
